AI tools are increasingly useful and we encourage group members to explore them. Our guidelines are broadly aligned with the [Leiden Declaration on Artificial Intelligence](https://leidendeclaration.ai/), which we recommend reading.

# Disclosure

Any substantive AI contribution to a paper must be disclosed. Follow the target journal's policy. When in doubt, disclose. AI tools must not be listed as authors.

# Writing

AI assistants can help with editing and clarity. Do not use AI to generate scientific content you have not independently verified. AI models frequently hallucinate references — always check that a suggested citation exists and supports the claim.

# Code and data analysis

AI coding assistants are generally lower risk since code can be tested, but always review AI-generated code carefully and test it on cases where you know the answer.

# Peer review

Do not submit confidential manuscripts to AI tools without checking whether the journal permits this.
